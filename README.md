# 🚀 CogniCode – Adaptive Cognitive Coding Platform

## 🧠 About the Project
CogniCode is an intelligent coding platform designed to improve programming skills through adaptive learning.  
It tracks user behavior such as attempts, time spent, and performance to provide personalized guidance.

---

## 🔥 Features
- ✨ Futuristic UI (Glassmorphism + Neon Theme)
- 🧠 Adaptive Learning System
  - Hint after 1 wrong attempt
  - Solution after 2 wrong attempts
  - Similar Question after 4 wrong attempts
- 💻 Interactive Coding Workspace
- 📊 Analytics Dashboard (Time, Speed, Performance)
- 🔁 Multi-question flow
- 🔐 Login system (demo validation)

## ▶️ How to Run the Project

### Step 1: Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/cognicode.git
cd cognicode
```

### Step 2: Install dependencies
```bash
pip install -r requirements.txt
```

### Step 3: Run the application
```bash
python app.py
```

### Step 4: Open in browser
```
http://127.0.0.1:5000
```
